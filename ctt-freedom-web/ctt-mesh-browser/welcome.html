Getting Started

  1. Install the CTT Mesh Network daemon:
    $ sudo dnf install ctt-mesh-network-1.0-1.fc42.x86_64.rpm
  2. Start the daemon:
    $ ctt_mesh_daemon &
  3. Verify it's running:
    $ curl http://localhost:8765/status
  4. Navigate to content:

    Click the extension icon and enter a content hash, or click any ctt:// link on the web.

How to Use

Method 1: Extension Popup

  • Click the CTT Mesh icon in your browser toolbar
  • Enter a 64-character SHA-256 hash
  • Click "Navigate" to retrieve content from the mesh

Method 2: Direct URLs

  • Enter a URL like: ctt://a1b2c3d4... in the address bar
  • The extension will automatically intercept and retrieve the content

Method 3: Click CTT Links

  • Any ctt:// links on web pages will be enhanced with a green indicator
  • Click them to navigate through the mesh network

Publishing Content

To share files on the CTT Mesh Network:

# Publish a file $ ctt_mesh_publish -f myfile.html # Publish a directory $ ctt_mesh_publish -d /path/to/website/ # Content hash will be returned - share this hash!
